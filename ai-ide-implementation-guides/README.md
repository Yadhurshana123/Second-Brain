# AI IDE Implementation Guides

This folder contains prompt guides for implementing features using the 2nd Brain documentation.

Files:

- [[backend-imple]] — backend feature implementation prompt and required reading order.
- [[frontend-imple]] — frontend feature implementation prompt and required reading order.

Use these files before asking Cursor or another AI IDE to implement a feature.

The AI must read the required documentation first, summarize understanding, then implement only the requested feature.
